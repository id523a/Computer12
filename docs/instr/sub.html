Instruction: sub

sub Rd, Rs
csub Rd, Rs
cond00110Rd (3 bits, 0-7)Rs (3 bits, 0-7)

cond10110Rd (3 bits, 8-15)Rs (3 bits, 0-7)

Action

Takes two operands: a destination register Rd and a source register Rs.
The value of Rs is subtracted from Rd, and the result is stored back in Rd. The K flag is set to 1 if a borrow was required and 0 otherwise. The values are treated as unsigned integers, or equivalently, as 2's complement signed integers. All arithmetic operations 'wrap around' on overflow.

If the instruction is conditional (cond = 1), and the P flag is 0, the instruction has no effect.

Flags

  • Z: Set to 1 if the subtraction result (not including the borrow) was zero, and set to 0 otherwise.
  • S: Set to the most significant bit of the subtraction result.
  • K: Set to 1 if a borrow was required and 0 otherwise.
  • V: Set to 1 if 2's complement overflow occurred during the subtraction, and to 0 otherwise.
  • P: Unchanged.
