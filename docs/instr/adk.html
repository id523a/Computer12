Action

Takes two operands: a destination register Rd and a source register Rs.
The values of Rd, Rs, and the K flag (0 or 1) are added together, and the result is stored in Rd. The K flag is set to 1 if a carry was generated and 0 otherwise. The values are treated as unsigned integers, or equivalently, as 2's complement signed integers. All arithmetic operations 'wrap around' on overflow.

If the instruction is conditional (cond = 1), and the P flag is 0, the instruction has no effect.

Flags

  • Z: Set to 0 if the addition result (not including the carry out) was non-zero, and unchanged otherwise.
  • S: Set to the most significant bit of the addition result (not including the carry).
  • K: Carry into and out of the addition.
  • V: Set to 1 if 2's complement overflow occurred during the addition, and to 0 otherwise.
  • P: Unchanged.
