Instruction: ror

Action

These instructions take one operand: a destination register Rd.
The instructions perform a circular shift of the bits in Rd, by one place.
rol shifts the bits to the left, i.e. towards the most significant bit. The MSB of the old value becomes the LSB of the new value.
ror shifts the bits to the right, i.e. towards the least significant bit. The LSB of the old value becomes the MSB of the new value.

If the instruction is conditional (cond = 1), and the P flag is 0, the instructions have no effect.

Flags

  • Z: Set to 1 if the value in Rd is zero, and set to 0 otherwise.
  • S: Set to the most significant bit of Rd after the bit rotation.
  • K: Unchanged.
  • V: Set to an unspecified value.
  • P: Unchanged.
