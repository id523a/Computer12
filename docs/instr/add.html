Instruction: add

add Rd, Rs
cadd Rd, Rs

Action

Takes two operands: a destination register Rd and a source register Rs.
The values of Rd and Rs are added together, and the result is stored in Rd. The K flag is set to 1 if a carry was generated and 0 otherwise. The values are treated as unsigned integers, or equivalently, as 2's complement signed integers. All arithmetic operations 'wrap around' on overflow.

If the instruction is conditional (cond = 1), and the P flag is 0, the instruction has no effect.

Flags

  • Z: Set to 1 if the addition result (not including the carry) was zero, and set to 0 otherwise.
  • S: Set to the most significant bit of the addition result (not including the carry).
  • K: Carry out from the addition, if values are treated as unsigned.
  • V: Set to 1 if 2's complement overflow occurred during the addition, and to 0 otherwise.
  • P: Unchanged.
