Action

These instructions take one operand: a destination register Rd.
The instructions shift the bits in Rd by one place. The K flag takes the value of the bit that is shifted out.
shl performs a left shift. 0 is shifted into the LSB.
shr performs a logical right shift. 0 is shifted into the MSB.
asr performs an arithmetic right shift. The bit that is shifted into the MSB is a copy of the existing MSB.

If the instruction is conditional (cond = 1), and the P flag is 0, the instructions have no effect.

Flags

  • Z: Set to 1 if the value in Rd is zero after the bit shift, and set to 0 otherwise.
  • S: Set to the most significant bit of Rd after the bit shift.
  • K: Set to the bit that is shifted out from Rd.
  • V: Set to an unspecified value.
  • P: Unchanged.
