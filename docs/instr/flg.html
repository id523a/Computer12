Instruction: flg

flg N
cflg N
cond11001101N (3 bits, 0-7)

Action

This instruction takes 1 immediate 3-bit operand, N, ranging from 0 to 7.
It causes the next N executed instructions to only affect processor flags, and to not write their results to any registers, or to memory. Any load or store instructions that occur within the next N instructions will be treated as a no-op.

In combination with arithmetic instructions, this instruction is useful for performing numeric comparisons or bit testing without overwriting the values being compared. cflg can also be used to create conditional loads and stores.

If the instruction is conditional (cond = 1), and the P flag is 0, this instruction has no effect. That is, subsequent instructions will execute normally.

Flags

  • All flags are unaffected.
