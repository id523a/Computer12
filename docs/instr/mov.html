Instruction: mov

mov Rd, Rs
cmov Rd, Rs
cond00000Rd (3 bits, 0-7)Rs (3 bits, 0-7)
cond01000Rd (3 bits, 0-7)Rs (3 bits, 8-15)
cond10000Rd (3 bits, 8-15)Rs (3 bits, 0-7)

Action

Takes two operands: a destination register Rd and a source register Rs.
The value of Rs is copied into Rd.

If the instruction is conditional (cond = 1), and the P flag is 0, the instruction has no effect.

Flags

  • All processor flags are unaffected.
