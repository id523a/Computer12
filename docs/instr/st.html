Instruction: sta/stb/stc/std

sta [location]
stb [location]
stc [location]
std [location]
1111Rs (2 bits)Location (6 bits)

Action

Takes two operands: a source register Rs, and a location.
The source register must be one of A, B, C, or D.
The value in the register Rs is stored at the specified location.

Unlike with most instructions, load and store instructions do not have a conditional counterpart. However, cflg can be used to conditionally execute this instruction.

Flags

  • All processor flags are unaffected.
