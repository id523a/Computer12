Action

These take two operands: a destination register Rd and a source register Rs.
These instructions compute the bitwise AND/OR/XOR of Rd and Rs, and store the result in Rd.

If the instruction is conditional (cond = 1), and the P flag is 0, the instruction has no effect.

Flags

  • Z: Set to 1 if the result of the bitwise operation was zero, and set to 0 otherwise.
  • S: Set to the most significant bit of the result.
  • K: Unchanged.
  • V: Set to an undefined value.
  • P: Unchanged.
