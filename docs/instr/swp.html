Instruction: swp

swp Rd
cswp Rd
cond11001100Rd (3 bits, 0-7)

Action

This instruction takes one operand: a destination register Rd. Rd must be a 'near' register.
The instruction swaps the most significant 6 bits of Rd with the least significant 6 bits. Equivalently, it performs a circular rotation of Rd by 6 bits. It can be used to separately manipulate the upper and lower halves of a data word.

If the instruction is conditional (cond = 1), and the P flag is 0, this instruction has no effect.

Flags

  • Z: Set to 1 if the value in Rd is zero, and set to 0 otherwise.
  • S: Set to the most significant bit of Rd after the swap.
  • K: Unchanged.
  • V: Set to an unspecified value.
  • P: Unchanged.
